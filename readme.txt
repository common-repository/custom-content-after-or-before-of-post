== Description ==
Insert custom content after or before or both of the post. Support HTML


== Installation ==
1. Download the plugin and extract its content
2. Upload the complete folder `yahoo-skype-status` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to  Settings --> Custom content after or before of post
